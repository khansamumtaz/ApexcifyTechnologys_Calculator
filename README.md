# Calculator
A stylish and responsive **Premium Calculator** built using **HTML, CSS, and JavaScript**.  
This calculator features a clean neumorphic design, smooth button interactions, keyboard support, and all essential arithmetic operations.

## Features
- ➕ Addition
- ➖ Subtraction
- ✖ Multiplication
- ➗ Division
- 📊 Percentage Calculation
- 🧹 Clear Screen (`C`)
- ⌫ Delete Last Character
- 🕘 Expression History Display
- ⌨ Keyboard Support
- 📱 Responsive Design for Mobile & Desktop
- 🎨 Modern Premium User Interface

## Technologies Used
- **HTML5**
- **CSS3**
- **JavaScript**

## How to Run This Project
### Method 1: Run Directly
1. Download the project files.
2. Open the folder.
3. Double-click on **`index.html`**.
4. The calculator will open in your browser.
   
### Method 2: Run from GitHub
1. Clone this repository:
https://github.com/khansamumtaz/ApexcifyTechnologys_Calculator/tree/main

2. Open the project folder.
3. Run `calculator.html` in your browser.

## Functionalities

This calculator allows users to:

- Enter numbers and operators
- Perform calculations instantly
- View previous calculation in the history display
- Delete the last entered value
- Clear the full screen
- Use keyboard keys for faster input
  
## Keyboard Support
The following keys are supported:
| Key | Function |
|-----|----------|
| `0 - 9` | Number Input |
| `+` | Addition |
| `-` | Subtraction |
| `*` | Multiplication |
| `/` | Division |
| `%` | Percentage |
| `.` | Decimal |
| `Enter` or `=` | Calculate Result |
| `Backspace` | Delete Last Character |
| `Esc` or `C` | Clear Screen |
